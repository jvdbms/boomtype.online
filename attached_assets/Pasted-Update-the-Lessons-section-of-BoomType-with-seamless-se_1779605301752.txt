SPACE / ENTER KEY TRANSITION RULES
═══════════════════════════════════════

RULE 1 — AUTO-DETECT SESSION COMPLETE
When user types the last character of any round:
  → Immediately show a brief result overlay
  → Display: WPM | Accuracy | Errors
  → Show message: "Press SPACE or ENTER for next round"
  → Do NOT show any button that requires mouse click

RULE 2 — SPACE OR ENTER STARTS NEXT
When user presses SPACE or ENTER on result screen:
  → Instantly dismiss result overlay
  → Load next round content immediately
  → Move cursor to first character automatically
  → User can start typing right away
  → Zero delay — transition under 300ms

RULE 3 — PHASE TRANSITION (Letter → Word → Paragraph)
When Round 10 of any phase completes:
  → Show phase complete screen for 1 second
  → Display: "Phase 1 Complete! Moving to Word Practice..."
  → Show animated progress: ✓ Letter  →  [Word]  →  Paragraph
  → After 1 second auto-countdown: "Starting in 3... 2... 1..."
  → OR user can press SPACE/ENTER to skip countdown
  → Next phase loads instantly

RULE 4 — LESSON TRANSITION
When all 3 phases of a lesson complete:
  → Show lesson complete celebration (brief animation)
  → Display: "Lesson 1 Complete! Home Row Mastered ✓"
  → Show stats summary: Total WPM average, Total accuracy
  → Show: "Press SPACE to start Lesson 2 — Upper Row"
  → Press SPACE → next lesson Phase 1 Round 1 starts

RULE 5 — RETRY WITH KEYBOARD
If user wants to retry current round:
  → Press R key to retry same round instantly
  → Press SPACE/ENTER to move forward
  → Show hint at bottom: "[SPACE] Next  [R] Retry  [ESC] Menu"
